\chapter*{Conclusion Générale}
\addcontentsline{toc}{chapter}{Conclusion Générale}

Ce Projet de Fin d'Année (PFA) a permis de concevoir et développer \textbf{Stockcare}, une application répondant concrètement aux besoins de gestion de stock dans le secteur médical.

\section*{Travail accompli}
Nous avons mis en place une architecture complète allant de la base de données relationnelle au frontend interactif. Les fonctionnalités clés telles que la gestion des lots et la traçabilité des mouvements sont opérationnelles.

\section*{Perspectives}
Pour la suite, nous envisageons :
\begin{itemize}
    \item L'intégration de lecteurs de codes-barres physiques pour accélérer les saisies.
    \item Un module de prédiction des commandes basé sur l'historique de consommation (IA).
    \item Une application mobile pour les magasiniers.
\end{itemize}

Ce projet a été une excellente opportunité de monter en compétence sur la stack Spring Boot / Next.js et la méthodologie de développement d'applications d'entreprise.
