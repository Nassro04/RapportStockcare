\chapter*{Résumé}
\addcontentsline{toc}{chapter}{Résumé}

Stockcare est une solution complète de gestion de stock pharmaceutique conçue pour optimiser le suivi des médicaments, depuis leur réception jusqu'à leur dispensation. Ce projet combine une architecture backend robuste en Java (Spring Boot) et une interface utilisateur moderne en React (Next.js).

L'objectif principal est de réduire les erreurs de stock, d'automatiser les alertes de péremption et de fournir une traçabilité complète des mouvements.

\vspace{2cm}

\textbf{Mots-clés :} Gestion de Stock, Spring Boot, Next.js, PostgreSQL, Docker, Microservices, Pharmacie.
