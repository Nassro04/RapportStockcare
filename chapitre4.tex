\section{Exemple de Code Frontend}
Voici un extrait du composant React gérant le formulaire de sortie de stock :

\begin{lstlisting}[language=Javascript, caption=Extrait de stock-exit-form.tsx]
// Extrait de code représentatif (simplifié)
const StockExitForm = () => {
  const { register, handleSubmit } = useForm();
  
  const onSubmit = async (data) => {
    try {
      await api.post('/stock/exit', data);
      toast.success('Sortie enregistrée');
    } catch (error) {
      toast.error('Erreur lors de la sortie');
    }
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      {/* Champs du formulaire */}
    </form>
  );
};
\end{lstlisting}
